# Como usar
Copie o arquivo de variáveis de ambiente e em seguida edite o arquivo com os valores necessários

```
cp .env.example .env
```

Instalando dependências
```
npm install
```

Ambiente de desenvolvimento
```
npm run dev
```

Build
```
npm run build
```

Executando código a partir do build
```
npm start
```
